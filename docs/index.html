Add the store, then install

Add the Island Bitcoin marketplace to your node once — every app appears inside.

🟣 Umbrel

  1. App Store → ⋯ → Add community store
  2. Paste the store URL (click to copy)
  3. Install any Island Bitcoin app

🔵 Start9

  1. System → Marketplace → Add custom registry
  2. Paste the registry URL (click to copy)
  3. Install any Island Bitcoin app
